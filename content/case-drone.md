# Drone AI System

Edge-first architecture for aerial image analytics. Designed path-planning, image telemetry, and a mock inference pipeline suitable for deployment on low-power GPUs and TPU-compatible hosts.

**Core Technologies**
- TensorFlow & Keras: Model training and inference
- YOLO: Real-time object detection
- Embedded Systems: Low-power hardware integration
- Satellite Data: Imagery processing and analysis
- Path-Planning Algorithms: Optimized flight routes for data collection
