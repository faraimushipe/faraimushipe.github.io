Drone AI System

Edge-first architecture for aerial image analytics

Overview

This project designed a path-planning and inference pipeline for drones, optimized for low-power GPUs and suitable for real-time aerial analysis.

Key Components

  • Path-Planning: Algorithms for efficient flight routes to maximize data collection.
  • Image Telemetry: Real-time transmission and processing of aerial imagery.
  • Inference Pipeline: Mock setup for AI model deployment on edge devices.

Technologies Used

TensorFlow & Keras for model development, YOLO for detection, embedded systems for hardware integration, and satellite data APIs.

Challenges & Solutions

Balancing computational power with battery life required custom optimizations. We used TPU-compatible models for efficiency.

Future Applications

This system can be extended to wildlife monitoring, search and rescue, and urban planning.
