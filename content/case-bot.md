# WhatsApp Automation Bot

Built production-grade chatbots using Botpress and BotPenguin. Integrated with n8n to automate support flows and CRM updates. Deployed to campus with 150+ active users, automating ~70% of common requests.

**Core Technologies**
- Botpress & BotPenguin: Chatbot development and deployment
- API Integration: WhatsApp Business API, CRM systems
- NLP: Natural language processing for user interactions
- Cloud Hosting: Scalable deployment for high user volume
- Automation: n8n workflows for backend processes
