Building a university radar that maps motion

Published Sep 2024 | IoT, Radar Technology, Motion Detection

This project focused on developing a radar system for university campuses to detect and map motion, enhancing security and research capabilities.

Objective

Create an affordable, accurate motion-mapping radar for educational environments.

Implementation

Used ultrasonic sensors and Doppler radar integrated with IoT platforms for real-time mapping.

Key Features

  • Real-time motion tracking
  • Data visualization on web dashboards
  • Integration with university security systems

Applications

Security monitoring, research in physics, and student projects in IoT.
