# 4Thoughts — Premium Portfolio

This repo is a premium, production-ready static portfolio for Farai Mushipe (4Thoughts).
Deploy to GitHub Pages by pushing contents to a repository and enabling Pages from main branch root.

Contact form uses Google Apps Script — deploy Code.gs from the Apps Script editor and set script properties:
- ADMIN_EMAIL = your email
- ALLOWED_ORIGINS = https://yourusername.github.io

Place the Apps Script web app URL in index.html as window.GOOGLE_SCRIPT_URL or edit main.js to set it.
